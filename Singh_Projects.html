Scratch Project

CSP

Fall 2018

Play the Game

This project involved the use of scratch, a visual programming language available for everyone on the internet. Through this project we solidified our knowledge on basic programming principles such as variables by implementing them into a project of our choice. With scratch, we were allowed to make a game or a story, of which we chose a game modeled after the popular Space Invaders. Our game consisted of 2 levels, one normal and one boss level, both of which would contain enemies randomly shooting the player, the only separation being that the boss ship would move in directions while the enemies in level 1 would stay unmoving. Through this project we learned fundamental programming concepts such as variable types and operators. In addition, we also learnt organization practices for a project, such as keeping multiple versions of the product, regularly updating a daily log, and to design a product in three tiers, with more features and higher workload in each tiers, which would keep our goals clear and our progress easily trackable.
